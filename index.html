Algunos de los proyectos en los que participé

E-commerce de Moda "Estilo Urbano"

Desarrollo de una plataforma de comercio electrónico completa para una tienda de ropa urbana. Implementación de un catálogo de productos con filtros avanzados, carrito de compras, pasarela de pago segura y gestión de pedidos. Tecnologías utilizadas: React, Node.js, MongoDB.

Aplicación de Gestión de Tareas "TaskMaster"

Creación de una aplicación web para la gestión de tareas personales y profesionales. Funcionalidades principales: creación, edición y eliminación de tareas, asignación de prioridades, recordatorios y visualización de progreso. Tecnologías utilizadas: Angular, Firebase.

Blog de Viajes "Explora el Mundo"

Descripción: Diseño y desarrollo de un blog de viajes interactivo y visualmente atractivo. Implementación de un sistema de gestión de contenidos (CMS) personalizado para la publicación de artículos, galerías de fotos y videos. Tecnologías utilizadas: WordPress (con tema personalizado), PHP, MySQL.

Plataforma de Aprendizaje Online "Code Academy"

Desarrollo de una plataforma de aprendizaje en línea para cursos de programación. Funcionalidades principales: gestión de usuarios, creación de cursos con lecciones interactivas, foros de discusión y sistema de evaluación. Tecnologías utilizadas: Python (Django), PostgreSQL, JavaScript.

Sitio Web Corporativo "Innovate Solutions"

Diseño y desarrollo de un sitio web corporativo moderno y profesional para una empresa de consultoría tecnológica. Implementación de secciones informativas sobre los servicios ofrecidos, casos de éxito, equipo de trabajo y formulario de contacto. Tecnologías utilizadas: HTML, CSS, JavaScript, Bootstrap.

Aplicación Móvil de Recetas "Chef Digital"

Creación de una aplicación móvil para la búsqueda y gestión de recetas de cocina. Funcionalidades principales: búsqueda por ingredientes, categorías y dificultad, creación de listas de compras, temporizador de cocción y comentarios de usuarios. Tecnologías utilizadas: React Native, Firebase.
